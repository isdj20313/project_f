# 요구사항 명세서 (Software Requirements Specification)

**프로젝트명**: 벤포드 법칙 데이터 검증기 (Benford's Law Detector)
**버전**: 1.0.0
**작성일**: 2026-06-17

---

## 1. 개요

### 1.1 목적
숫자 데이터의 첫 유효숫자 분포를 벤포드 법칙과 비교하고, 카이제곱 적합도 검정으로
해당 데이터가 자연스러운 분포를 따르는지("정상") 혹은 벗어나는지("조작 의심 신호")를
판정하는 소프트웨어를 개발한다.

### 1.2 배경 및 교과 연계
- **확률과 통계**: 확률분포(벤포드 분포), 가설검정(카이제곱 적합도 검정)
- **수학(로그·지수)**: 상용로그를 이용한 첫자리 확률 유도 및 첫자리 추출

### 1.3 사용자
- 통계·데이터에 관심 있는 학생 및 일반 사용자
- 프로그래밍 지식이 없어도 대시보드로 체험 가능해야 한다.

---

## 2. 전체 처리 흐름

```
[1] 숫자 데이터 입력
      ↓
[2] 첫 유효숫자(1~9) 추출 (0·비숫자 제외)
      ↓
[3] 실제 첫자리 분포(빈도·비율) 계산
      ↓
[4] 벤포드 이론 분포와 비교 (표·그래프)
      ↓
[5] 카이제곱 적합도 검정 수행
      ↓
[6] "정상 / 조작 의심" 판정 및 결과 출력
```

## 5. 입출력 명세

### 5.1 입력
- 내장 예시 데이터 (이름으로 선택)
- 텍스트 붙여넣기 (줄바꿈/쉼표/공백 구분)
- CSV 파일 업로드 (분석할 열 선택)
- CLI: `--sample`, `--csv --column`, `--file`

### 5.2 출력
- 첫자리별 관측수/관측%/벤포드% 비교 표
- 관측 vs 벤포드 막대그래프
- 카이제곱 통계량, 임계값, p-value, MAD
- 판정 배너: ✅ 정상 / 🚨 조작 의심 / ⚠️ 표본 부족

---

## 6. 검증(판정) 기준

- 귀무가설 H0: "데이터의 첫자리 분포는 벤포드 분포를 따른다."
- 검정통계량: `chi^2 = Σ (O_d - E_d)^2 / E_d` (자유도 8)
- 판정:
  - `chi^2 ≤ 임계값(α)` → H0 기각 못 함 → **정상**
  - `chi^2 > 임계값(α)` → H0 기각 → **조작 의심 신호**
  - 표본 < 30 → 검정 신뢰도 낮음 경고

- 여러 데이터셋 일괄 비교 리포트 생성
